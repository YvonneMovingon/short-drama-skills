# 04 · 按剧情连贯拆分副本

**适合场景：** 长剧本分集制作 / 多集连续短剧  

基于叙事焦点、空间一致性、动作连贯性和情绪流向，将连续 Clip 聚合为 Shot，并处理合并争夺与防孤立逻辑。适用于长剧本分集时保持故事节奏不断裂。

**[在 LuxReal 中一键使用 →](https://luxreal.com?skill=04)**

---

## 完整 Skill

```markdown
## 一、叙事和人物关联聚合

### 判定对象：
不同 Clip 是否围绕**同一叙事**展开。

### 说明：
比如有 clip1、2、3、4，clip1和2是叙事X，clip3和4是叙事Y，则clip1和2聚合，clip3和4聚合。

### 叙事中核心人物关系稳定性规则
1. 若人物增减属于同一叙事的自然进退场，可合并（群像对峙 → A 单人特写怒斥）
2. 若新增人物带来新的冲突关系或话题焦点，必须切分（A+B 争执 → C 出现参与）
3. 若互动对象发生根本变化（A 对 B → A 转而攻击 C），应优先切分
4. 若核心主体未变化，且空间未变化，默认优先合并

### 快速闪回片段合并规则
- 若 clip 是闪回片段，且时长低于2s，优先合并且合并到主体前面的回忆片段

### 其他规则
1. 若叙事发生外向性实质性转移，开启新的 Shot；若只是同一人物在同一事件链条中的状态变化（如昏迷→醒来），不视为焦点转移，可继续聚合
2. 即使人物数量增减、出现单人反应特写，只要事件核心未转移，可以继续聚合

---

## 二、场景空间一致性

### 判定对象：物理空间与时间是否连续

### 必须切分情况：
- 室内 → 室外 / 地点变化 / 明确时间跳跃 / 光线环境显著变化 / 空间结构改变

即使叙事焦点未变，只要空间变化，必须切分。

### 补充：如果是闪回导致的场景空间变化，无须切分。

---

## 三、动作连贯性

若多个 Clip 属于同一连续动作链，应优先聚合：
- 推搡 → 摔倒 → 爬起
- 冲出屋 → 拉扯 → 被甩开

只要不违反前两条规则，可合并。

---

## 四、情绪/视线一致性

若情绪保持同一主导方向，可优先合并（持续争吵、持续压迫、持续哀求）

若情绪发生明显断裂式转折，应考虑切分：
- 愤怒 → 突然冷静谈判
- 绝望 → 突然大笑
- 紧张对峙 → 突然温情拥抱

正反打或视线匹配镜头可优先合并：
- 人物看向窗外（主观）→ 窗外的风景（客观）
- 人物开始回忆 → 具体闪回片段

---

## 全局主动合并 Shot 规则
- **主动合并原则**：相邻两个片段相加后的总时长严格 ≤ 15秒，且符合前述连贯性标准，即可主动合并
- **合并争夺处理**：当一个 Clip 既可以和前一个合并也可以和后一个合并时，进行双向关联度比对，优先划入叙事动作更不可分割的那一组
- **防孤立机制**：避免将某个 Clip 合并后导致另一侧变成极其零碎的孤立短镜头

---

## 执行步骤
1. 通读输入的 Clip
2. 按照上述规则进行聚合
3. 对完成聚合的 Shot，再执行全局主动合并规则
```

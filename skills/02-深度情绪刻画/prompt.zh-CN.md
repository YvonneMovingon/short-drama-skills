# 一、对话拆分的核心逻辑（总纲）
## 对话不是语言交流，而是——
- 权力博弈
- 信息转移
- 情绪攻防
- 关系重构

# 二、对话拆分核心原则

## ① 权力变化触发切镜（Power Shift Cut）
### 必切条件：
- 信息优势转移（A占上风 → B反杀）
- 立场改变 / 揭露隐瞒信息 / 关系重新定义 / 情绪控制权转移
### 执行方式：
- 权力上升者 → 微仰拍 + 特写
- 权力下降者 → 微俯拍 + 反应特写
- 群体压迫 → 利用前景遮挡体现压迫感
### 禁止：连续"平视中景A ↔ 平视中景B"重复乒乓。

## ② 信息单位拆分原则（Information Unit Split）
对话拆分单位 ≠ 一整句台词，而是信息单位（Information Beat）
每个镜头只能承载一个核心信息点：一个事实/一个态度/一个指控/一个反问/一个决定
如果一句台词包含多个信息点 → 必须拆分

## ③ 反应优先原则（Reaction First Rule）
反应镜头的重要性 ≥ 说话镜头
### 必须插入反应镜头的情况：
- 听到爆炸信息 / 情绪被刺穿 / 被揭穿 / 被羞辱 / 被背叛
### 反应必须外化为：
- 手部动作（指节发白、攥衣角）/ 呼吸变化（胸口起伏、屏住呼吸）/ 目光游移（瞳孔微缩）/ 下意识动作
- 禁止"面无表情看着对方"

## ④ 对话节奏断点原则（Rhythm Break Rule）
可作为断点的元素：倒水 / 点烟 / 走位改变 / 拉开距离 / 站起 / 转身 / 拿出关键道具
当动作改变空间关系 → 必须切镜

## ⑤ 视角层级递进原则
对话冲突升级时必须执行景别递进，不是横向平切
推荐节奏：中景（关系建立）→ 过肩 → 近景 → 特写 → 极近特写（爆点）
> 情绪越近，景别越近。禁止在高潮时还停留在中景。

---
# 三、画面描述与执行基准

## ① 人物生理与设定一致性
- 生病/重伤/虚弱的角色必须体现出吃力、喘息、强撑的体感，绝对禁止出现"呼吸平稳、步伐矫健"等与设定相悖的描述

## ② 画面绝对具象化
- 禁止使用剧作分析视角的抽象概念词
- 强制转化为可见细节：所有权力关系必须翻译成摄影机能拍到的五官肌肉、肢体幅度、空间距离
  - 错误：徐有田搂住徐长寿，宣布主导权
  - 正确：徐有田一步跨上前，左臂死死勒住徐长寿的肩膀将他半挡在身后，下巴微扬，死盯对面的徐来顺

## ③ 动作与调度的物理连贯性
- 发生走位、保护动作时，必须先交代空间位置的变化，再接动作爆点

---
# 四、对话类型专项拆分规则

## 1 争吵型对话（Argument）
- 语速快 / 权力频繁波动
- 单句最长不超过6秒 / 高频短镜头（2–4秒）/ 多用机位高度变化

## 2 情感对话（Confession / Emotional）
- 情绪渐变 / 表达压抑或坦白
- 不频繁切镜 / 允许长镜头（但不超过12秒）/ 情绪点必须给特写

## 3 审问型对话（Interrogation）
- 控制与被控制
- 控制方微仰拍 / 被审问方俯拍 / 适当加入沉默空白

## 4 交易型对话（Negotiation）
- 表面平静 / 潜台词强
- 必须加入道具特写 / 必须外化潜台词 / 避免纯正反打

---
# 五、对话镜头结构模板（标准结构）
适用于2人对话：
1. 建立关系镜头（中景）
2. 说话者镜头（主动）
3. 聆听者反应镜头（权力变化）
之后才允许进入乒乓结构。

---
# 六、禁止规则
- 违背人物生理设定
- 使用抽象剧作概念词代替具体画面描述
- 连续两个相同景别+相同机位高度+相同视角
- 连续三句台词不切镜
- 关键爆点没有特写
- 只拍说话者，不给反应
- 纯台词无肢体动作 / 用空镜代替反应
- 禁止遗漏台词 / 禁止改变台词顺序

---
# 七、对话镜头节奏控制
| 情况 | 推荐时长 |
|------|------|
| 普通对话 | 3–5秒 |
| 情绪爆点/强烈震荡 | 3秒 |
| 快速争吵 | 2–4秒 |
| 情绪渐变 | 6–8秒 |

---
# 八、长台词切分（触发条件：预估时长超过 8s）
**核心原则：零删减 + 有节奏。** 严禁压缩、省略、改写或合并原始台词。

切分锚点：话题转折点 → 情绪变化点 → 逻辑层级点 → 自然气口

视听手法同 Skill 01（T1–T7）

---
# 九、对话拆分本质公式
> 对话拆分 = (人物设定底色) × 信息单位 × 权力变化 × 情绪波动 × 空间关系 × 视觉具象化

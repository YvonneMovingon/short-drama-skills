## 完整 Skill

```markdown
# 动作戏情节拆分原则

# 一、动作与叙事双螺旋原则
- 优先拆解动作链，但不可略过"核心专有名词"、"特殊道具"与"剧情反转锚点"
- 若涉及"拜师誓言"、"金手指触发"、"核心转折台词"，动作必须为叙事高光让路或伴随执行

# 二、动作拆分分级原则（核心）
- **普通动作/过招（普攻）**：允许压缩在一个镜头中完成，鼓励用连贯动作减少切镜
- **关键动作（暴击/终结技/第一击）**：必须严格拆解为三段：动机/预备 → 动作执行 → 结果/反馈

# 三、动接动原则强化版
动作切点必须落在"运动轨迹进行中"，而非动作完成后
- 正确：刀刚抬到半空 → 切 → 刀挥下的一半
- 目的：保持动势连续，避免节奏塌陷

# 四、物理反馈原则（核心）
任何攻击动作必须有物理反馈（受力位移、身体变形、物体破碎等）
- **同框反馈（常规打斗）**：同一镜头内同时展现攻击与受击反应
- **特写反馈（重击/终结）**：单给受击方特写镜头，强调破坏力

# 五、动态运镜与空间连贯（核心）
- **位移闭环合并**：奔跑、跳跃、翻滚等位移，优先使用"动态跟拍镜头"，将起步-过程-抵达合并在一个镜头中
- 仅在跨越极大空间或需要强调落点时，才拆分为起跳与落地两镜

# 六、节奏分层原则
动作戏节奏必须分层：
1. 爆发段（快切/连贯猛攻）
2. 观察段（2秒缓冲对峙）
3. 反制段（节奏再拉升）
结构建议：慢 → 快 → 更快 → 突停 → 再爆发

# 七、关键动作独占镜头
以下动作必须独占镜头：
- 金手指/底牌触发（掏出体验卡、捏碎符箓、点击系统面板）
- 关键反杀 / 第一滴血 / 武器亮相 / 角色首次爆发

# 八、受伤视觉确认
如果剧本写到受伤、流血，必须有"伤口确认镜头"（血从指缝渗出、衣服染红），禁止只用表情代替伤害表现

# 九、动作戏禁止纯对话对打
长台词不允许两人站定对骂超过5秒。必须采用：
- 动作伴随：边走位、防御、抛掷道具边说话
- 情绪缓冲极点：激烈交锋后的短暂安静时刻给面部特写

# 十、群体打斗拆分规则
3人以上打斗需区分：主线对抗、辅助干扰、远处威胁
镜头焦点只能集中在一条动作线上，禁止全员乱斗模糊画面

# 十一、切镜阈值（提高标准）
动作方向改变、站位交替等，**优先通过镜头内部调度解决**
必须切镜的触发条件精简为：
- 主导权发生重大逆转
- 武器损毁更替
- 受到重创
- 场景转移

# 十二、动势压迫镜头原则
角色被压制时：机位必须降低（仰拍施压者）或升高（俯拍被压制者），不允许平视表达权力失衡

# 十三、动作高潮延长原则
最终击倒/终结一击：允许使用慢动作（3–4秒），高潮动作时长必须比普通动作更长

# 十四、网文特有元素强制视觉化
凡涉及网文设定（系统提示/面板、系统奖励结算），必须以"悬浮UI/光幕特写"分配独立镜头视觉化呈现

# 十五、心理情绪反应镜头
动作的威力不仅是物理破坏。必须为受击方或旁观者分配情绪反应镜头（底牌爆发时脸上狂妄转惊恐的微表情特写）

# 优先级冲突原则
1. 核心叙事锚点完整性（关键剧情/系统道具/反转瞬间绝不遗漏）
2. 角色核心情绪反馈与关键台词传达
3. 动势连续性与单镜完整度（优先保证流畅，减少废镜）
4. 物理反馈完整性
5. 节奏冲击

## 分镜时长估算（动作戏专用）

| 镜头类型 | 时长 |
|---------|------|
| 微动作（眨眼、转头、举杯） | 2秒 |
| 常规动作（站起、开门、走几步） | 2秒 |
| 复杂连续动作（武打一套、翻墙） | 4秒 |
| 强烈情绪震荡 | 3秒 |
| 慢动作（关键终结技） | 3~4秒 |
| 时长上限 | **15秒以内** |
```
